In this case, the number of beats on all chords can be "scaled" to fit the new time signature. This means that the bars will be the same. Select the second option above if this is what you want. One or more chords have bets which can't be automatically "scaled" to fit the new time signature. It's still fine to change the time signature, but the chords will keep their current number of beats. This means that the bars will be different.
