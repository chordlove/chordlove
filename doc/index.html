Chordlove documentation

Chordlove is client-side application written in JavaScript (and HTML5 + CSS). The source code is found at GitHub.

The core code is contained in modules loaded by Inject. This code mostly handles the infrastructure of the application.

Most funcionality lives in plugins, handled by the plugins module.

Dependencies

  • Font Awesome
  • Inject
  • JQuery
  • JQuery UI
  • Twitter Bootstrap

Fonts

  • DejaVu Font
  • Neuton Cursive Font
